# 贡献指南

感谢你考虑为《Crossroads: Rebirth》做出贡献！

## 行为准则

请保持尊重、包容的沟通方式...

## 如何贡献

### 报告Bug
1. 使用Bug报告模板创建Issue
2. 清晰描述复现步骤
3. 提供环境信息

### 提交新功能
1. 先创建Feature Request Issue讨论
2. 获得维护者反馈后再开始开发
3. 遵循开发规范

### 代码贡献流程
1. Fork 本仓库
2. 从 `develop` 创建功能分支: `git checkout -b feature/your-feature`
3. 提交你的改动: `git commit -m 'feat: 添加某个功能'`
4. 推送到分支: `git push origin feature/your-feature`
5. 创建Pull Request到 `develop` 分支

## 开发环境搭建
1. 安装 Godot 4.5.1
2. 克隆仓库
3. 打开 `project/project.godot`

## 代码审查
所有PR需要至少1名维护者review通过才能合并
