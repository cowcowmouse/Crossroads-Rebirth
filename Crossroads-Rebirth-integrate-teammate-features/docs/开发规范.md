# 《Crossroads: Rebirth》开发规范

## 1. 代码规范

### 命名规则
- **文件/类名**：snake_case.gd (如 `game_manager.gd`)
- **变量/函数**：snake_case (如 `current_ap`, `consume_action_point()`)
- **常量**：SCREAMING_SNAKE_CASE (如 `MAX_ACTION_POINT`)
- **信号**：snake_case (如 `action_point_changed`)
- **节点路径**：PascalCase (如 `UIManager/Control/ActionPointLabel`)

## 2. Git规范

Commit规范

docs - 文档更新
